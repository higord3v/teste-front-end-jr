Desafio front-end realizado desenvolvendo uma interface de loja, os principais desafios foram a implementação de carrossel e modal de produtos.

## Tecnologias e técnicas

- Vite
- React.js
- Sass
- Axios
- Figma
- Componentização

## Pré-requisitos

- Node.js
- pnpm
- npm
- yarn

## Instalar dependências

- pnpm install
- npm install
- yarn add

## Rodar projeto

- pnpm run dev
- npm run dev
- yarn dev
